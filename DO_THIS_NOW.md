## In GitHub Desktop:

1. **Look at the TOP RIGHT** of GitHub Desktop window
2. **Click the button that says "Publish branch"** (even if it looks grayed out, try clicking it)
3. **If nothing happens or it asks to sign in:**
   - Click **"Repository"** menu at the top
   - Click **"Push"**
   
4. **If that doesn't work:**
   - Click **"Branch"** menu at the top  
   - Click **"Push Branch"** or **"Push Origin"**

5. **If it asks for sign in:**
   - Sign in with your GitHub account
   - Click "Allow" or "Authorize"

6. **Done!** Your code is on GitHub.

---

## Still Not Working?

**Last resort - super simple:**

1. In GitHub Desktop, click **"Repository"** menu
2. Click **"View on GitHub"** 
3. This opens GitHub in your browser
4. On GitHub website, click **"Set up in Desktop"** button
5. It will open GitHub Desktop and should work
