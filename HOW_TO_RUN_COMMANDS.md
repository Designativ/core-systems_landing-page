## Quick Guide:

### Where to Run Commands:

**In Cursor (the app you're using now):**

1. **Open Terminal in Cursor:**
   - Look at the **bottom** of your Cursor window
   - Click on the **"Terminal"** tab
   - OR press `` Ctrl+` `` (Control + backtick key)
   - OR press `` Cmd+` `` on Mac (Command + backtick)

2. **You'll see a terminal window** at the bottom of Cursor

3. **Type your commands there** and press Enter

---

## What You See:

The terminal will look like this:

```
nataliiaivanova@Nataliias-MacBook-Pro Website 2.0 %
```

This is where you type commands!

---

## Example - Deploy to Vercel:

1. **Open terminal** (bottom of Cursor)

2. **Type this command and press Enter:**
   ```bash
   npm install -g vercel
   ```
   - Wait for it to finish (takes ~10 seconds)

3. **Then type:**
   ```bash
   vercel
   ```
   - Press Enter
   - It will open a browser to log in
   - Follow the prompts

4. **Done!** Your site deploys!

---

## Common Terminal Commands:

- `pwd` - Shows where you are (current folder)
- `ls` - Lists files in current folder
- `cd /path/to/folder` - Change to that folder
- `npm install` - Install packages
- `git push` - Push code to GitHub

---

## Tips:

- **Type carefully** - commands are case-sensitive
- **Press Enter** after typing each command
- **Don't close terminal** while commands are running
- **Wait for completion** - look for the prompt to return (shows `%`)
